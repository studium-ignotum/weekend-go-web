## Overview
[refactor]: Theme 2 — extract HTML partials, centralize SVG icons, data-driven carousel

### Goal
Eliminate ~700 LOC duplication: 4 HTML pages share single source-of-truth header/footer/CTA/icons; carousel slides driven from JSON; APK URL lifted into `package.json` version. Build pipeline gains lightweight HTML partial inclusion via custom node script.

### Context
AUDIT-REPORT Theme 2 đã liệt kê: nav/footer/CTA copy-pasted qua 4 pages với drift, star SVG ×15 (~500 LOC), FAQ chevron ×5, APK URL hardcoded ×2, carousel slides ×7. Single source = better DX, prevents brand drift.

### Requirements
- [REQ-01] Tạo `partials/header.html` chứa canonical nav (legal-page slim variant + index mobile menu)
- [REQ-02] Tạo `partials/footer.html` chứa canonical footer (full version, dynamic copyright)
- [REQ-03] Tạo `partials/download-cta.html` chứa App Store + Google Play buttons với placeholder `{{APK_VERSION}}`
- [REQ-04] Tạo `partials/icons.svg` chứa `<defs>` + 10 `<symbol>` (star, faq-chevron, app-store, google-play, social-fb, social-ig, linkedin, mail, hamburger, carousel-prev, carousel-next)
- [REQ-05] Replace 35+ inline SVG occurrences với `<svg><use href="#icon-X"/></svg>` references
- [REQ-06] Move 4 HTML files vào `src/*.src.html`; output `*.html` ở root sinh ra bởi build script
- [REQ-07] Tạo `scripts/build-html.js`: process `<!-- @include partials/X.html -->` markers + `{{APK_VERSION}}` placeholder
- [REQ-08] Update `package.json`: extend `build` script chạy `node scripts/build-html.js` SAU tailwind compile; thêm `dev` watch nếu cần
- [REQ-09] Update `tailwind.config.js`: thêm `partials/**/*.html` + `src/**/*.src.html` vào `content` array
- [REQ-10] Lift APK URL: `package.json` thêm field `appVersion: "1.0.35"`; build script inject vào `{{APK_VERSION}}` placeholder
- [REQ-11] Carousel slides → JSON inline `<script type="application/json" id="carousel-slides">`; `carousel.js` đọc + render
- [REQ-12] Sau refactor: `git diff` showing root `*.html` files có content equivalent (chỉ khác ở: dynamic copyright year, h4 typos không tái xuất hiện, color tokens unified)

## Implementations

### Design Decisions
| # | Decision | Reasoning | Type |
|---|----------|-----------|------|
| 1 | Custom node script over `posthtml-include` / `eleventy` | 4 pages, 1 build step — no need for framework. ~30 LOC, no deps | LOCKED |
| 2 | Source HTMLs renamed `*.src.html`, output ở root | Giữ deploy structure, không đổi serve path; file extension makes source/output relation rõ ràng | LOCKED |
| 3 | Partials chứa thẻ `<nav>` / `<footer>` / `<section>` only — NOT `<head>` | Mỗi page có meta SEO/OG riêng | LOCKED |
| 4 | Icons.svg inlined ở đầu `<body>` mỗi page (hidden via `style="display:none"`) | `<use>` references có thể trỏ tới `<symbol>` cùng document — không cần fetch | LOCKED |
| 5 | Carousel slides: inline JSON trong index.src.html, không file external | Tránh extra HTTP request; size nhỏ | LOCKED |
| 6 | APK_VERSION placeholder syntax: `{{APK_VERSION}}` (Mustache-like) | Distinctive, không clash với HTML/JS | LOCKED |
| 7 | Build script idempotent: chạy 2 lần ra identical output | Safety: nếu user accidentally chạy build trên output, output không bị recursive include | LOCKED |
| 8 | Build script fail loudly if partial missing | Catches typos | LOCKED |

### Snippet examples

**`scripts/build-html.js` (~30 LOC):**
```js
const fs = require('fs');
const path = require('path');
const pkg = require('../package.json');

const sources = fs.readdirSync('src').filter(f => f.endsWith('.src.html'));
for (const src of sources) {
  const name = src.replace('.src.html', '.html');
  let html = fs.readFileSync(path.join('src', src), 'utf8');
  html = html.replace(/<!--\s*@include\s+(\S+)\s*-->/g, (_, partial) => {
    const p = path.join(__dirname, '..', partial);
    if (!fs.existsSync(p)) throw new Error(`Missing partial: ${partial} in ${src}`);
    return fs.readFileSync(p, 'utf8');
  });
  html = html.replace(/{{APK_VERSION}}/g, pkg.appVersion);
  fs.writeFileSync(name, html);
  console.log(`✓ ${name}`);
}
```

**Source HTML usage:**
```html
<body>
  <!-- @include partials/icons.svg -->
  <!-- @include partials/header.html -->
  <main>
    ...
    <!-- @include partials/download-cta.html -->
    ...
  </main>
  <!-- @include partials/footer.html -->
</body>
```

**Star reference (replaces 33 lines):**
```html
<svg class="w-5 h-5 text-yellow-400" aria-hidden="true"><use href="#icon-star"/></svg>
```

**Carousel JSON:**
```html
<script type="application/json" id="carousel-slides">
  [
    {"src":"assets/images/news-feed.webp","alt":"News feed app"},
    {"src":"assets/images/weekend-plan.webp","alt":"Weekend plan"},
    ...
  ]
</script>
```

`carousel.js` chuyển từ `document.querySelectorAll('.phone-item')` sang đọc JSON + render `<div class="phone-item">` dynamically.
